# Animation of Squares

## 📖 Overview
This repository contains a simple website that demonstrates creative **CSS animations** applied to squares.  
It is built entirely with **HTML and CSS**, showcasing how pure CSS can be used to create smooth transitions, transformations, and engaging visual effects without JavaScript.

## ✨ Features
- Animated squares using CSS `@keyframes`
- Smooth transitions and transformations
- Lightweight and responsive design
- Easy to customize colors, sizes, and animation speed

## 🛠️ Technologies Used
- **HTML5** – for structure
- **CSS3** – for styling and animations

## 🚀 How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Run the `index.html` file in any modern browser.
